# 📊 Tech Gadget Usage and Mental Wellbeing Analysis

## 📌 Project Overview
This project analyzes the relationship between tech gadget usage and mental wellbeing using Python.

## 🔍 Objectives
- Analyze screen time impact on sleep
- Study productivity vs gadget usage
- Understand physical activity impact on mental health

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📈 Key Insights
- Higher screen time is slightly associated with reduced sleep
- Active individuals show better mental health scores
- Moderate gadget usage shows balanced wellbeing

## 📂 Dataset
CSV dataset containing:
- Screen Time
- Sleep Hours
- Mental Health Status
- Productivity Score
- Physical Activity
